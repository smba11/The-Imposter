## Play in the browser
This project now includes a browser-friendly version you can host with GitHub Pages.

1. Push this repository to GitHub.
2. In your repository settings, enable GitHub Pages and point it at the root (or the `main` branch).
3. Visit the published URL to play.

The browser game lives in `index.html` and `web.js`.

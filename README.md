## Play in the browser
This project includes a browser-friendly version you can host with GitHub Pages.

You can deploy from either location:

- **Root** (`/`): use the files at the repo root (`index.html`, `web.js`).
- **Docs** (`/docs`): use the same files inside `docs/` (`docs/index.html`, `docs/web.js`).

### GitHub Pages setup
1. Push this repository to GitHub.
2. In your repository settings, enable GitHub Pages.
3. Choose either the root (`/`) or `/docs` folder as the source.
4. Visit the published URL to play.
